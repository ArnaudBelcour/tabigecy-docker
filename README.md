# Docker recipe to build an image containing EsMeCaTa and Tabigecy

This repository contains a Dockerfile recipe to build an image containing EsMeCaTa and Tabigecy. It is build based on micromamba-docker.

The Image also contains EsMeCaTa precomputed database and ete4 preinstanciated with the version of the NCBI Taxonomy database used to generate the precomputed database.
